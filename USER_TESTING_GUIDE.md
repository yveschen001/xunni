# XunNi Bot - MBTI 功能用户测试指南

> **测试环境**: Staging  
> **Bot**: @xunni_bot (Staging)  
> **状态**: ✅ 已通过自动化测试，准备用户测试

---

## 🎯 测试目标

验证新的 MBTI 对话式流程是否符合用户期望，包括：
1. 注册时的 3 个 MBTI 选项
2. 12 题对话式测验
3. `/mbti` 命令管理

---

## 📱 如何开始测试

### 方式 1: 新用户注册（推荐）

1. 打开 Telegram
2. 搜索 `@xunni_bot`（Staging 版本）
3. 点击 **Start** 或发送 `/start`
4. 按照提示完成注册

### 方式 2: 清除现有数据重新测试

如果你已经注册过，需要清除数据：
1. 联系管理员清除你的测试账号
2. 或使用新的 Telegram 账号

---

## ✅ 测试场景清单

### 场景 1: 手动输入 MBTI（约 2 分钟）

**步骤**：
1. [ ] 发送 `/start` 开始注册
2. [ ] 选择语言（中文繁体或英文）
3. [ ] 输入昵称（任意）
4. [ ] 选择性别（男/女）
5. [ ] 确认性别
6. [ ] 输入生日（格式：YYYY-MM-DD，如 1995-06-15）
7. [ ] 确认生日
8. [ ] **看到 3 个选项按钮**
9. [ ] 点击 **"✍️ 我已经知道我的 MBTI"**
10. [ ] **看到 16 个 MBTI 类型按钮**
11. [ ] 选择任意类型（如 INTJ）
12. [ ] **看到 MBTI 描述**
13. [ ] 完成反诈骗确认
14. [ ] 同意服务条款
15. [ ] **注册完成**

**检查点**：
- [ ] 3 个选项按钮清晰可见
- [ ] 16 个 MBTI 类型按钮排列整齐
- [ ] 选择后显示类型描述
- [ ] 流程流畅，无卡顿
- [ ] 文案清晰易懂

---

### 场景 2: 进行 MBTI 测验（约 3 分钟）

**步骤**：
1. [ ] 发送 `/start` 开始注册
2. [ ] 完成语言、昵称、性别、生日
3. [ ] **看到 3 个选项按钮**
4. [ ] 点击 **"📝 进行快速测验"**
5. [ ] **看到第 1 题**（共 12 题）
6. [ ] **看到进度条**（▓░░░░░░░░░ 8%）
7. [ ] 点击任意答案
8. [ ] **自动进入第 2 题**
9. [ ] 依次回答所有 12 题
10. [ ] **看到测验结果**（如 ESTP）
11. [ ] **看到类型描述**
12. [ ] 完成反诈骗和服务条款
13. [ ] **注册完成**

**检查点**：
- [ ] 每题显示清晰
- [ ] 进度条正确更新
- [ ] 题号显示正确（1/12, 2/12...）
- [ ] 答案按钮响应快速
- [ ] 测验结果准确
- [ ] 描述文字完整

**测验问题示例**：
- Q1: 在社交场合中，你通常：
  - 主动与他人交谈
  - 等待他人来找我
- Q2: 周末你更喜欢：
  - 和朋友出去玩
  - 在家独处休息
- ... (共 12 题)

---

### 场景 3: 跳过 MBTI（约 2 分钟）

**步骤**：
1. [ ] 发送 `/start` 开始注册
2. [ ] 完成语言、昵称、性别、生日
3. [ ] **看到 3 个选项按钮**
4. [ ] 点击 **"⏭️ 稍后再说"**
5. [ ] **看到提示**："你可以随时使用 /mbti 指令来设定"
6. [ ] 完成反诈骗和服务条款
7. [ ] **注册完成**（MBTI 为空）

**检查点**：
- [ ] 跳过选项清晰可见
- [ ] 提示信息友好
- [ ] 可以正常完成注册
- [ ] 不强制要求 MBTI

---

### 场景 4: 使用 `/mbti` 命令（约 3 分钟）

**前提**：已完成注册（可以是跳过 MBTI 的用户）

**步骤**：
1. [ ] 发送 `/mbti` 命令
2. [ ] **看到当前 MBTI 状态**
   - 如果已设定：显示类型和来源（手动/测验）
   - 如果未设定：显示"你还没有设定 MBTI 类型"
3. [ ] **看到菜单选项**：
   - 📝 重新进行测验
   - ✍️ 手动输入 MBTI
   - 🗑️ 清除 MBTI（如果已设定）
   - ❌ 取消

**测试 A: 重新测验**
4. [ ] 点击 "📝 重新进行测验"
5. [ ] 回答 12 题
6. [ ] **看到新的测验结果**
7. [ ] 发送 `/mbti` 验证更新

**测试 B: 手动修改**
8. [ ] 发送 `/mbti`
9. [ ] 点击 "✍️ 手动输入 MBTI"
10. [ ] 选择新类型（如 ENFP）
11. [ ] **看到更新确认**
12. [ ] 发送 `/mbti` 验证更新

**测试 C: 清除 MBTI**
13. [ ] 发送 `/mbti`
14. [ ] 点击 "🗑️ 清除 MBTI"
15. [ ] **看到清除确认**
16. [ ] 发送 `/mbti` 验证已清除

**检查点**：
- [ ] 状态显示准确
- [ ] 来源标记正确（手动/测验）
- [ ] 菜单选项完整
- [ ] 所有操作正常工作
- [ ] 取消按钮正常关闭

---

### 场景 5: 中断恢复（约 2 分钟）

**步骤**：
1. [ ] 开始 MBTI 测验
2. [ ] 回答前 3 题
3. [ ] **关闭 Bot**（不要完成测验）
4. [ ] 等待 10 秒
5. [ ] **重新打开 Bot**
6. [ ] 发送任意消息或 `/start`
7. [ ] **应该从第 4 题继续**

**检查点**：
- [ ] 测验进度正确保存
- [ ] 重新进入时从断点继续
- [ ] 之前的答案已记录
- [ ] 进度条正确显示
