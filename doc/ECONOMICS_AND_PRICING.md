# 💰 經濟模型與定價策略 (Economics & Pricing)

本文件記錄專案的代幣經濟模型、定價策略與成本分析。

## 1. 貨幣匯率 (Exchange Rates)

我們基於以下匯率進行定價計算：

*   **基準貨幣**：USDT / USD
*   **平台貨幣**：Telegram Stars (XTR)
*   **匯率**：
    *   **1 USDT ≈ 50 Telegram Stars** (官方充值匯率參考，約 $0.02/Star)
    *   **實際變現匯率**：Telegram 官方提現手續費較高，開發者實際收入可能低於此比例。
    *   **用戶購買成本**：視地區而異，約 $0.02 - $0.03 / Star。

> **注意**：1 USDT = 77 Stars 是特定渠道或批量購買的優惠匯率，或是為了簡化計算的內部參考值。在此文檔中，我們以 **1 USDT = 50 Stars** 作為保守估計的開發者收入基準，以 **1 USDT = 77 Stars** 作為用戶視角的性價比參考。

*(註：您提到的 1 USDT = 77 Stars 可能是指某些特定市場的購買力，我們在定價時需考慮這個波動。)*

## 2. 促銷定價策略 (Promotional Pricing Strategy)

為了提高轉化率，我們採用 **「高標價、大折扣」** 的展示策略。

### 2.1 價格顯示規則 (Display Rules)

*   **倍率**：所有項目的「原價」設定為實際售價的 **5倍**。
*   **訊息顯示 (Message)**：同時顯示原價（刪除線）與特價，強調 **"-80% OFF"**。
*   **按鈕顯示 (Button)**：**僅顯示實際支付價格** 加上「特價」標籤，避免用戶對支付金額產生混淆。
    *   ✅ 正確：`🔥 特價 385 Stars`
    *   ❌ 錯誤：`~1925~ 385 Stars` (避免在支付按鈕上顯示干擾資訊)

## 4. 單次付費功能與虛擬道具 (Pay-Per-Use & Items)

我們引入「算命瓶」作為核心消耗道具。

### 4.1 算命瓶 (Fortune Bottle) 獲取

#### 每週配額 (Weekly Free)
*   一般用戶：每週補滿至 **1 瓶**。
*   VIP 用戶：每週補滿至 **7 瓶**。

#### 付費充值 (Top-up)

**方案 A：小包 (10 瓶)**
*   **實際售價**: **100 Stars**
*   **顯示原價**: **500 Stars**

**方案 B：大包 (50 瓶) - 推薦**

    > 💰 **算命瓶大包 (50個)**
    > 原價 ~~1925 Stars~~
    > **特價 385 Stars** 🔥 (-80% 超值)

*   **裂變獎勵**: 邀請的朋友(B)再邀請朋友(C)時 -> **A 獲得 +1 算命瓶**。

## 6. 成本控制策略 (Cost Control)

我們採用 **Hybrid AI Architecture** 以平衡成本與品質。

### 6.1 模型分層策略

*   **Tier 1: 複雜推理與創意 (Complex Reasoning)**
    *   **任務**: 深度算命 (Ziwei, Astrology, Match)、多步驟推理鏈 (Chained Generation)。
    *   **主要模型**: `Gemini 1.5 Pro` / `Gemini 2.0 Flash` (Google Cloud)
    *   **備援模型**: `GPT-4o` (OpenAI) - 當 Google API 失敗時自動切換。
    *   **成本**: 中高。

*   **Tier 2: 快速回應與翻譯 (Fast Response)**
    *   **任務**: 漂流瓶翻譯、簡單對話、意圖識別。
    *   **主要模型**: `Gemini 1.5 Flash` / `Gemini 2.0 Flash-Lite`
    *   **備援模型**: `GPT-4o-mini`
    *   **成本**: 極低 (接近免費)。

### 6.2 成本估算

*   **算命成本估算**:
    *   單次深度算命 API 成本 (混合): ~$0.01 - $0.03
    *   最低充值單價收入: ~$0.15
    *   **最低毛利**: ~80%

*   **免費用戶**:
    *   嚴格限制使用 `Gemini Flash` 或 `GPT-4o-mini`。
    *   今日運勢：簡短版。

*   **VIP 用戶**:
    *   優先調用 Tier 1 模型 (`Gemini Pro` / `GPT-4o`) 提供最佳品質。
    *   免費獲得 7 瓶/週，價值約 50-70 Stars。
