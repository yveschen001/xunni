---

### 📝 編輯暱稱

1. 點擊「📝 編輯暱稱」
2. 系統會提示：
   ```
   📝 **編輯暱稱**
   
   請輸入新的暱稱：
   
   💡 提示：
   • 最多 36 個字符
   • 顯示時最多 18 個字符
   • 避免廣告或不當內容
   ```
3. 直接輸入新暱稱，例如：`新暱稱123`
4. 系統會回覆：`✅ 暱稱已更新為：新暱稱123`

---

### 📖 編輯簡介

1. 點擊「📖 編輯簡介」
2. 系統會提示：
   ```
   📖 **編輯個人簡介**
   
   請輸入你的個人簡介：
   
   💡 提示：
   • 最多 200 個字符
   • 可以包含興趣、愛好等
   • 不允許包含外部連結
   ```
3. 直接輸入簡介內容
4. 系統會回覆：`✅ 個人簡介已更新！`

---

### 🌍 編輯地區

1. 點擊「🌍 編輯地區」
2. 系統會提示：
   ```
   🌍 **編輯地區**
   
   請輸入你的地區：
   
   💡 提示：
   • 例如：台北、高雄、台中
   • 最多 50 個字符
   ```
3. 直接輸入地區名稱
4. 系統會回覆：`✅ 地區已更新！`

---

### 🏷️ 編輯興趣

1. 點擊「🏷️ 編輯興趣」
2. 系統會提示：
   ```
   🏷️ **編輯興趣標籤**
   
   請輸入你的興趣標籤（用逗號分隔）：
   
   💡 提示：
   • 例如：音樂, 電影, 旅行, 美食
   • 最多 5 個標籤
   • 每個標籤最多 20 個字符
   ```
3. 輸入興趣，用逗號分隔
4. 系統會回覆：`✅ 興趣標籤已更新！`

---

### 💝 匹配偏好

1. 點擊「💝 匹配偏好」
2. 系統會顯示選項：
   ```
   💝 **匹配偏好**
   
   選擇你想要配對的性別：
   
   [👨 男生]
   [👩 女生]
   [🌈 任何人都可以]
   [↩️ 返回]
   ```
3. 點擊選擇
4. 系統會回覆：`✅ 匹配偏好已更新`

---

### 🩸 編輯血型

1. 點擊「🩸 編輯血型」
2. 系統會顯示選項：
   ```
   🩸 **編輯血型**
   
   💡 血型可用於 VIP 血型配對功能
   
   請選擇你的血型：
   
   [🩸 A 型] [🩸 B 型]
   [🩸 AB 型] [🩸 O 型]
   [不確定 / 暫不填寫]
   [↩️ 返回]
   ```
3. 點擊選擇
4. 系統會回覆：`✅ 血型已更新為 🩸 A 型`

---

### 🧠 重新測試 MBTI

1. 點擊「🧠 重新測試 MBTI」
2. 系統會開始 MBTI 測試（12 題）
3. 完成後會更新您的 MBTI 結果

## 🚫 常見問題

### Q: 為什麼點擊「編輯資料」沒反應？

**A: 請確認：**
1. 您使用的是 Staging Bot：@xunni_dev_bot
2. Bot 已更新到最新版本（Version ID: 62394f23-b05d-42b6-bbb0-b6840aefc56c）
3. 您已完成註冊流程

**解決方法：**
- 發送 `/start` 重新啟動 Bot
- 確認您的註冊狀態：發送 `/profile` 查看

---

### Q: 為什麼有些項目不能修改？

**A: 以下項目註冊後不可修改：**
- 👤 性別（永久不可修改）
- 🎂 生日（永久不可修改）

**可以重新測試的項目：**
- 🧠 MBTI（可重新測試）

**可以隨時修改的項目：**
- 📝 暱稱
- 📖 個人簡介
- 🌍 地區
- 🏷️ 興趣標籤
- 💝 匹配偏好
- 🩸 血型

---

### Q: 編輯後如何確認更新成功？

**A: 兩種方式確認：**

1. **查看系統回覆**
   - 成功：`✅ 暱稱已更新為：...`
   - 失敗：`❌ ...`

2. **重新查看個人資料**
   - 發送 `/profile`
   - 檢查更新的項目是否顯示新內容
