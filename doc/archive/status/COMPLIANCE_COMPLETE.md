## 🚀 **部署步驟**

### **Step 1: 部署 HTML 文件**

**選項 A：使用 Cloudflare Workers Sites（推薦）**

1. 配置 `wrangler.toml`：
```toml
[site]
bucket = "./public"
```

2. 部署：
```bash
pnpm deploy:staging  # 測試
pnpm deploy:production  # 生產
```

**選項 B：使用 Cloudflare Pages**

```bash
npx wrangler pages deploy public --project-name=xunni-legal
```

### **Step 2: 更新代碼中的 URL**

1. 創建 `src/config/legal_urls.ts`（配置文件）
2. 更新 3 個 Handler 文件的 URL 引用
3. 測試所有鏈接

**詳細步驟請參考：`LEGAL_DOCUMENTS_SETUP.md`**

### **Step 3: 設置 BotFather**

在 Telegram BotFather 中設置：

```
/setdescription
XunNi - MBTI 漂流瓶匿名交友 Bot

/setabouttext
（完整介紹，包含法律文檔鏈接）

/setcommands
（完整命令列表）
```

**詳細內容請參考：`LEGAL_DOCUMENTS_SETUP.md`**
